## Features

- Dark/Light Mode toggle
- GitHub Stats integration
- Responsive design
- Contact modal
- Typing effect animation
- Availability badge
- Microsoft Clarity analytics
- Cookie consent (Klaro)
- PWA support

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Framer Motion
- NextUI

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
